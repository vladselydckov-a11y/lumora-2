# Lumora client UI

Новый клиентский интерфейс поверх рабочего backend v7.4.

Что изменено:
- убраны технические названия, версия проекта и лишняя разработческая статистика из интерфейса;
- добавлен бренд Lumora как имя AI-аналитика;
- добавлены тёмная и светлая темы;
- убраны фотографии блюд, чтобы не показывать ненастоящие изображения;
- главный экран показывает только управленческие показатели: выручка, чеки, гости, средний чек, скидки, задачи и сигналы;
- вкладка настроек позволяет редактировать тему, план, цель среднего чека, лимит скидок, видимость карточек и компактный режим;
- сохранение настроек работает через localStorage и сразу обновляет интерфейс;
- backend /api/summary и lib/supabaseServer.js оставлены как рабочий слой реальных данных Supabase.

Важно:
- iiko → n8n → Supabase не переносится в интерфейс. Автоматизация остаётся отдельным слоем.
- Фудкост скрыт по умолчанию, потому что себестоимость iiko ещё не подключена.

# Resto Mini App v7.2 — что добавлено

v7 сохраняет все экраны v6 и добавляет слой Real Data MVP.

## Новые вкладки

- `Импорт` — preview загрузки CSV/Excel выгрузки iiko.
- `AI-анализ` — короткий аудит по потерям, точкам, команде и меню.
- `Задачи` — Action Tracker: рекомендация превращается в задачу с ответственным и сроком.
- `Демо` — режим показа клиенту без технического шума.
- `Неделя` — возвращена вкладка недельной динамики, которая была в коде, но не была доступна из меню.

## Новые API routes

- `/api/network`
- `/api/actions`
- `/api/import/preview`

## Важное

Импорт CSV пока работает как безопасный preview-шаблон. Для настоящего импорта нужно добавить parser CSV/Excel и upsert в Supabase таблицы:

- `daily_sales`
- `dish_sales`
- `waiter_sales`
- `ai_reports`

Не подключай iiko напрямую во frontend. Правильная схема: iiko → n8n/backend → Supabase → Mini App.
